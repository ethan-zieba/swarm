# TTICHAN GraSP CloG

## Project Summary

Traeffik - Reverse Proxy for accessing the services\n
T.O.M - Tracking & Observation Microagent\n
InfluxDB - For indexing the N.A.D.I.R logs\n
Clavister - Physical firewall\n
H.U.G.O - Highly-compressed Upload for Guaranteed Offsite storage\n
Ansible - For provisionning the Pi 4's\n
N.A.D.I.R - Network Anomaly Detection and Intrusion Reporter\n

Grafana - For the InfluxDB logs visualisation\n
Swarm - Multi-nodes cluster for orchestrating docker containers\n
Portainer - Graphical interface for managing the Swarm\n

Cloudflare - For creating a tunnel\n
GlusterFS - Replicated and distributed storage\n
